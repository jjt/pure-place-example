Default Buttons

To create a Pure Button, add the place-button classname to any <a> or <button> element.

Disabled Buttons

To mark a button as disabled, add the place-button-disabled classname alongside place-button.

Active Buttons

To style a button so that it appears "pressed", add the place-button-active classname alongside place-button.

Primary Buttons

Tables

Default Table

To style an HTML table, add the place-table classname. This class adds padding and borders to table elements, and emphasizes the header.

Bordered Table

To add horizontal and vertical borders to all cells, add the place-table-bordered classname to the <table> element.

# Make Model Year

Table with Horizontal Borders

If you prefer horizontal lines only, add the place-table-horizontal classname to the <table> element.

Striped Table

Large tables are easier to parse visually if rows are easily distinguishable. Adding the place-table-odd class name to every other <tr> element changes the background of the row and creates a zebra-styled effect.

To mark a link as disabled, add an <a> element with the place-menu-disabled class name. Disabled items appear faded and do not inherit hover styles.

Paginators

To create a paginator control, add the place-paginator classname to a <ul> element. For paginator controls that look like buttons, add the place-button class name. For a plainer paginator, you can omit this class.
